RESTful HTTP
Integration using RESTful HTTP focuses attention on the resources that are exchanged between
Bounded Contexts
, as well as the four primary operations: POST, GET, PUT, and DELETE. Many
find that the REST approach to integration works well because it helps them define good APIs for
distributed computing. It’s difficult to argue against this claim given the success of the Internet and
Web.
