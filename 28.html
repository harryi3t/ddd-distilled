Another Example: What Is a Flight?
In the airline industry, a “flight” can have multiple meanings. There is a flight that is
defined as a single takeoff and landing, where the aircraft is flown from one airport to
another. There is a different kind of flight that is defined in terms of aircraft maintenance.
And there is yet another flight that is defined in terms of passenger ticketing, either
nonstop or one-stop. Because each of these uses of “flight” is clearly understood only by
its context, each should be modeled in a separate
Bounded Context.
To model all three
of these in the same
Bounded Context
would lead to a confusing tangle.
